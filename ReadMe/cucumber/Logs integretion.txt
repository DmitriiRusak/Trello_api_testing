For additional logs in a project was used library Log4j2.
The config file log4j2.xml is located at - src/test/resources/log4j2.xml
It is set to store logs in a directory logs.

For realization was used:
- logFactory class - for distributing logger entity.
- MyRestAssuredFilter - for capturing all necessary HTTP ditails
- cucumber event listener approach, represents equivalent of testNG testListener class and is located at - src/test/java/api/utils/CucumberEventListener.java,
- Cucumber hooks - analog @Before... / @After... annotations presented in a TestNG.
